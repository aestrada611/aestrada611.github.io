Among Us Imposter Game

A game where users must find the space with the imposter. In order to win the user must not be killed and find the imposter before a set number of turns go by. The skeleton uses HTML, styling is CSS, and the logic for the game is written out in JavaScript.

Wildlife Tracker API

This API was created using ruby on rails and allows the user to make instances of animal sighitings. This API has a has_many belong_to reltationship between animals and sightings. The model for animal has columns for common name, latin name, and kigdom for the species. The model for sightings has columns for longitude, latitude, and a date.
